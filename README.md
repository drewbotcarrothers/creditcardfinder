# Canadian Credit Card Finder

A web application to find and compare the best Canadian credit cards.

## Features

- Compare credit cards from major Canadian banks
- Filter by category (cash back, travel, low interest, etc.)
- Sort by rewards rates, annual fees, benefits
- Detailed card information and application links

## Project Structure

```
├── scraping/          # Credit card data scrapers
├── web/              # Next.js web application
├── data/             # Scraped card data
└── .github/workflows/# Automation
```

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
