# Credit Card Scrapers

Python scrapers to collect credit card data from Canadian banks.

## Setup

```bash
cd scraping
pip install -r requirements.txt
```

## Running Scrapers

```bash
python scrape_all_cards.py
```

## Supported Banks

- RBC
- TD
- CIBC
- Scotiabank
- BMO
- American Express Canada
- MBNA
- Tangerine
